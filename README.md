## SerpineBot

[![CodeFactor](https://www.codefactor.io/repository/github/luferro/serpinebot/badge)](https://www.codefactor.io/repository/github/luferro/serpinebot)

A multipurpose Discord bot.

## Technologies

-   Node.js: >= 16.9.0
-   Discord.js
-   Typescript
-   Mongoose

## Installation

```
npm install
```

## Usage

```
npm start
```
