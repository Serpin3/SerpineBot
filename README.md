## SerpineBot

[![CodeFactor](https://www.codefactor.io/repository/github/serpin3/serpinebot/badge)](https://www.codefactor.io/repository/github/serpin3/serpinebot)

A multipurpose Discord bot.

## Technologies
* Node.js: >= 16.9.0
* Discord.js: 13.6.0
* Typescript: 4.6.3
* Mongoose: 6.2.9

## Installation
```
npm install
```

## Usage
```
npm start
```
